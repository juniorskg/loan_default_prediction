# Loan Default Prediction

## Project Overview

The aim of this project is to use machine learning techniques to help banks and other financial institution predictic if a loan is likely to default or not. 

I aim to achieve it by analysing borrowers financial and demographic information in oder to identify patterns associated with loan default risk.

---

## Business Problem


Loan defaults can cause significant financial losses for lenders.  
By predicting the likelihood of default, financial institutions can make better lending decisions and reduce risk exposure.

## Dataset

The dataset contains loan applicant information such as:

- Loan amount
- Income
- Employment length
- Credit history
- Debt-to-income ratio
- Loan status (target variable)

### Target Variable

- `1` → Default
- `0` → No Default
